# 🎧 Earpiece Rental Bot

Telegram-бот для аренды микронаушников (RU/EN).

## Быстрый старт

### 1. Получить токен бота
- Открой @BotFather в Telegram
- Напиши `/newbot`
- Скопируй токен

### 2. Узнать свой Telegram ID
- Напиши @userinfobot
- Скопируй свой `Id`

### 3. Установить зависимости
```bash
pip install -r requirements.txt
```

### 4. Настроить переменные
```bash
cp .env.example .env
# Отредактируй .env — вставь BOT_TOKEN и ADMIN_CHAT_ID
```

### 5. Запустить
```bash
BOT_TOKEN=xxx ADMIN_CHAT_ID=yyy python bot.py
```

## Деплой на Railway / VPS
```bash
# Railway (бесплатно):
# 1. Загрузи папку на GitHub
# 2. Подключи репо в railway.app
# 3. Добавь переменные BOT_TOKEN и ADMIN_CHAT_ID в Settings → Variables
```

## Как работает

| Действие | Описание |
|---|---|
| Клиент нажимает "Забронировать" | Выбирает продукт → вводит имя/телефон/даты |
| Бот проверяет доступность | Если даты свободны — показывает итог и цену |
| Клиент подтверждает | Заявка сохраняется в `bookings.json` |
| Тебе приходит уведомление | Кнопка "Подтвердить" блокирует даты |
